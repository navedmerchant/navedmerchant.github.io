Brave API Key Setup Instructions

  1. Create a new Brave API account:
    https://api-dashboard.search.brave.com/register
  2. Go to Subscribe and create a new free API key:
    https://api-dashboard.search.brave.com/app/subscriptions/subscribe?tab=ai
  3. Go to API keys and copy your API key:
    https://api-dashboard.search.brave.com/app/keys
